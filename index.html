Sistema Operativo y Redes

Conceptos clave de informática y conectividad

1. ¿Qué es un Sistema Operativo?

Un sistema operativo (SO) es el software que gestiona el hardware y los recursos de una computadora. Actúa como intermediario entre el usuario y el hardware.

  • Funciones: administración de archivos, procesos, memoria, dispositivos.
  • Ejemplos: Windows, Linux, macOS, Android.
  • Tipos: monousuario, multiusuario, en tiempo real, distribuidos.

2. Tipos de Redes

Una red es un conjunto de dispositivos interconectados para compartir información y recursos. Se clasifican por su tamaño, topología y propósito.

  • LAN: Red de área local (oficinas, casas).
  • MAN: Red de área metropolitana.
  • WAN: Red de área amplia (Internet).
  • Topologías: estrella, bus, anillo, malla.

3. Protocolos de Comunicación

Son reglas que permiten la transmisión de datos entre dispositivos. Garantizan que la información sea enviada, recibida y comprendida correctamente.

  • TCP/IP: base del funcionamiento de Internet.
  • HTTP/HTTPS: navegación web segura.
  • FTP: transferencia de archivos.
  • SMTP/IMAP: envío y recepción de correos electrónicos.

4. Seguridad en Redes

La seguridad de redes busca proteger los datos y sistemas conectados frente a accesos no autorizados, ataques y pérdidas de información.

  • Firewall: filtra el tráfico de red.
  • Antivirus y antispyware: detectan y eliminan amenazas.
  • VPN: conexión segura y cifrada.
  • Contraseñas y autenticación: seguridad de acceso.

5. Sistemas de Numeración

Los sistemas de numeración son métodos para representar cantidades. En informática se usan distintos sistemas, principalmente el binario.

  • Decimal: base 10 (números que usamos cotidianamente).
  • Binario: base 2 (usado por las computadoras: 0 y 1).
  • Octal: base 8.
  • Hexadecimal: base 16 (usado en programación y direcciones de memoria).

Las conversiones entre estos sistemas son esenciales para entender cómo las computadoras procesan y almacenan datos.
